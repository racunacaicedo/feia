Conecta Google Analytics Data API

Para ver datos reales en tiempo automático necesitas configurar un Service Account de Google Cloud. Son 5 pasos, toma 10 minutos.

  1. Ve a console.cloud.google.com → crea un proyecto (o usa uno existente)
  2. Activa la API: Google Analytics Data API
  3. IAM y administración → Cuentas de servicio → Crear cuenta → descarga el JSON
  4. En GA4 → Admin → Administración de acceso → añade el email del service account como Lector
  5. Copia el JSON descargado a api/service-account.json y edita GA4_PROPERTY_ID en api/ga4.php

Property ID numérico: GA4 → Admin → Configuración de propiedad → ID de propiedad
